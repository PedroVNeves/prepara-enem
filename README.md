# Prepara ENEM

Sistema preparatório para o ENEM com simulados, banco de questões e (futuramente) correção de redação com IA.

## Visão geral

- **Simulados**: geração de simulados a partir de um banco de dados de questões do ENEM.
- **Questões**: banco de dados de questões do ENEM (a ser fornecido).
- **Correção de redação com IA**: funcionalidade futura para avaliar e dar feedback sobre redações.
- **Login multi-perfil**:
  - Escolas (administração/gestão de alunos e turmas)
  - Alunos vinculados a uma escola
  - Alunos individuais (sem vínculo institucional)

## Status
